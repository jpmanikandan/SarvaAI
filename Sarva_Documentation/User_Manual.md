# Sarva AI - User Manual

## 1. Introduction
Welcome to **Sarva AI**, your personal expert assistant. This manual guides you on how to use the chat and voice features effectively.

## 2. Interface Overview
- **Chat Area**: Displays the conversation history.
- **Input Box**: Type your questions here.
- **Send Button**: Submit your text query.
- **Microphone Icon**: Toggle voice recording.
- **Language Dropdown**: Select your preferred language for responses.

## 3. How to Use

### 3.1 Text Chat
1.  Type a question, e.g., *"What is the 22K gold rate in Madurai?"*.
2.  Press **Enter**.
3.  Sarva AI will detect that this is a **Finance** query and provide a live rate table.

### 3.2 Voice Chat
1.  Click the **Mic** icon (it turns red 🔴).
2.  Speak clearly: *"Tell me about Indian Contract Act."*
3.  Click the **Mic** icon again to stop.
4.  Sarva AI will process your audio and respond with both text and voice.

### 3.3 Changing Language
-   If you want the answer in **Tamil**, select "Tamil" from the dropdown. 
-   You can ask in English, and Sarva AI will translate the expert advice into Tamil for you.

## 4. Tips for Best Results
-   **Be Specific**: Instead of "Law", ask "What is the punishment for theft under IPC?"
-   **Check Source**: Sarva AI provides advice based on its training. Always consult real professionals for critical legal/medical decisions.
